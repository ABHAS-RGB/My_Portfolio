# Abhas Mishra - Personal Portfolio

Welcome to the repository for my personal portfolio! This is a clean, highly responsive, and interactive static website built entirely from scratch using pure, "vanilla" web technologies.

## 🚀 Live Demo
The portfolio is deployed and currently live on Vercel:
**[View Live Portfolio](https://myportfolio-blue-xi-27.vercel.app/)**

## 🛠️ Technology Stack
This project was intentionally built without heavy frameworks to maximize performance, control, and simplicity:
- **HTML5**: Semantic and accessible structure.
- **CSS3**: Custom premium styling featuring CSS variables, flexbox, grid, and advanced **Glassmorphism** UI effects.
- **Vanilla JavaScript**: Used for interactive elements including mobile menu toggling, smooth scrolling, and scroll-triggered micro-animations using the native `IntersectionObserver` API.

## ✨ Key Features
- **Glassmorphism Aesthetics**: High-end frosted glass effects on cards and UI elements.
- **Dynamic Scroll Animations**: Elements smoothly fade and slide into view as the user scrolls down the page.
- **Fully Responsive**: Flawlessly adapts to all screen sizes, from large desktop monitors to mobile phones.
- **Static & Fast**: Zero build steps required. It runs natively in any modern browser.

## 💻 How to Run Locally
Because this project uses vanilla web technologies, there are no dependencies to install (`npm install` is not required).

1. Clone the repository:
   ```bash
   git clone https://github.com/ABHAS-RGB/My_Portfolio.git
   ```
2. Navigate to the folder:
   ```bash
   cd My_Portfolio
   ```
3. Open `index.html` in your favorite web browser! 
   *(Alternatively, use the VS Code "Live Server" extension for hot-reloading while editing).*
